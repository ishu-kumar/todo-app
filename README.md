# ✅ To-Do App

A simple and interactive **To-Do App** built with HTML, CSS, and JavaScript. Easily add, remove, and mark tasks as complete.

## 🚀 Features
- ➕ Add tasks
- ✅ Mark tasks as completed
- ❌ Remove tasks
- 💾 Data persists using localStorage (optional)
